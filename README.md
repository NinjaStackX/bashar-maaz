 
A fully responsive, interactive personal portfolio website built with **vanilla JavaScript** and **pure CSS** (no frameworks or libraries).

## ✨ Features

- Fully responsive (mobile, tablet, desktop)
- Dark / Light mode toggle with user preference saved in `localStorage`
- Project filtering by category (HTML, CSS, JavaScript)
- Interactive contact form with client-side validation
- Sticky navigation bar with active section highlighting on scroll
- Smooth fade-in animations and lazy loading for images
- Accessible (ARIA labels, keyboard navigation)

## 🛠️ Technologies used

- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)

## 🧪 How to run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/pure-js-portfolio.git
```

1. Open index.html directly in any modern browser.
2. Or use a local server (e.g., Live Server in VS Code).
